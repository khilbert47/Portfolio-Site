This is Your Life Podcast Website

This website was created to be a hub for all content related to this podcast. Connecting with the host, find past and current resources, as well as joining the weekly email list.

Site Features

  • Episode Resources (links to articles, documents, etc.)
  • Display Current Reviews
  • About Host Page and Contact Resources
  • Play Recent Episode Feature

Process

Stage 1: Low Fidelity Design

The process started with a low-fidelity design of the layout for the website on paper to work through visual appearance, placement of elements, and basic basic visual heirarchy. This included creating and sketching four separate layouts for each of the main pages to the website:

  • Home Page
  • About the Host
  • Connect Page
  • References/Resources

Each separate page followed a similar design and feel and included a description of the main/most important content listed on the page.

Stage 2: HTML Layout and CSS

The second stage of this process included the beginning creation of the HTML documents and filling in missing elements from the orginal layout. These elements included an f.a.q. section on the about page and an episode player on the connect page.

The html document began with a very basic layout placing content and finding an even better layout than what was created in the beginning. As I moved into styling with CSS is when I realized more content was needed for this site. Placing an episode player on the connect page but also the home page was one the biggest moves made for this site so users can begin listening on the site itself before subscribing on their perferred platform. I placed it on both pages so finding the player takes less time and effort and is placed where you expect to find it.

Through CSS the naivgation was built. I wanted a responsive navigation that differed between a large and small viewport. With a larger viewport, 640px and larger, I created a navigation that displayed the different site pages at the top to better utlizie the space compared to a toggle navigation in a smaller viewport. Both navigations have animations within them when you hover over the different pages for better usability.

The style I decied to use for the homepage on this website was having a section linking to each page in the navigation. For users who know exactly what they are looking for the navigation will lead them right to that page but for users who are exploring their way across the site the homepage is their first journey. Through the homepage they ar given a quick overview of each page throughout the site and links to each.
