This is Your Life Podcast App

This app was created to be a hub for all content related to this podcast. Connecting with the host, find past and current resources, as well as joining the weekly email list.

App Features

  • Episode Resources (links to articles, documents, etc.)
  • Display Current Reviews
  • About Host Page and Contact Resources
  • Play Recent Episode Feature

Process

Stage 1: Low Fidelity Design

The process started with a low-fidelity design of a the layout for the website on paper to work through visual appearance, placement of elements, and basic basic visual heirarchy. This included creating and sketching four separate layouts for each of the main pages to the website:

  • Home Page
  • About the Host
  • Connect Page
  • References/Resources

Each separate page followed a similar design and feel and included a description of the main/most important content listed on the page.
