Just Stories Podcast

* Current Project - In Progress*

This website was created for the "Just Stories" Podcast. Here vistors can learn more about each member of the "Just Stories" team, explore a library of podcast episodes,and contact the team directly to become a future guest on the show.

Site Features

  • Contact Form
  • Smooth Scrolling
  • Scroll Box for Library of Current Podcast Episodes
  • Custom Graphics

Process

[Stage 1] Concept and Color Palette

The process began with identifying main prioriities for the website as well as a color palette. The clients vocalized that the elements most important to them were an about page and a contact form. The color palette was based off of two of the logo's primary colors and chosing slightly different shades for better visibility/color contrast.



[Stage 2] High Fidelity Design Using Adobe XD

After chosing a color palette I began creating a simple layout of elements using Adobe XD. The first version was a wireframe while the second version was branded and prototyped to flow through the site as a user would.

Interact with this wireframe here.



[Stage 3] HTML Layout and CSS

The third stage of this process is where the website began to come to life. Per the clients' request, the layout remained clean and simplistic with a fair amount of white space to reflect this minimalistic style. I chose a fixed header to keep the logo front and center and a contact form at the bottom of each page for easy access.

A problem I ran into early on was maintaining uniformity in a minimalistic design due to busy background in photos provided to me. To combat this conflict I decided to create custom graphics of each team member in a color blocking style to reduce any unwanted noise. I accomplished this by utilizing Procreate.

These graphics then became the icon and link to each biography for the team members. On the about page the user can click on each team member graphic and Javascript smooth scrolling will take them to each individual biography.

On the podcast page links are placed to each site where users can find the podcast to listen to. Each link uses a css animation to change color using the css "hover" psuedo-class. In addition to links, this page also contains a library of current episodes hosted in a scroll box to save space and add more straight lines to the page continuing the simplistic look.



What is Next?

Moving forward, I will be completing the contact form and it's backend development. Through PHP, I will be able to finalize this form for users' to be able to directly email the "Just Stories" team about becoming a possible future guest. I will also be adding in a section on the "Connect" page that will display featured Facebook comments for users' to be able to scroll through, similar to reviews.

Lastly I will be moving the website to a hosting website for the client, inserting any last needed content or graphics, setting up each media queries/break-points for all screen sizes including desktop, tablet, and phone, and running through accessablity checks for the website including font sizing, color contrasts, and alternative text.
